NodeJS-React website  
======================

Udemy ecommerce project

Tools and Software used:
------------------------
-   VSCode
-   Hyper
-   MongoDBCompass
-   Stripe
-   Firebase
-   Cloudinary

### Back-End:
-   NodeJS
-   Express
-   MongoDB

### Front-End:
-   ReactJS
-   Bootstrap
-   AntDesign
-   Redux

Main idea of the project:
-------------------------
-   Create, read, update and delete data
-   Sorting products
-   Rating system
-   Authentication
-   Purchasing products with card
